GameJolt


Game Jolt is a social community platform for video games, gamers, and content creators.


  • The Game Jolt Application Programming Interface (usually known as the GJAPI) allows any developer using a game development platform that supports HTTP operations and MD5 or SHA-1.


  • Game Jolt is a hosting service for freeware and commercial video games with social functions. It is where anyone can create a game, show it off, and build an audience.


  • In March 2022, Game Jolt launched a mobile application simultaneously on the Google Play Store and Apple App Store for Gen Z gamers and creators.
